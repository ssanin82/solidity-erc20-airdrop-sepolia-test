## Setup
You need Foundry to run the code.

The private key to my wallet is saved as "mykey" in Foundry's keystore.

Also:
```.env
SEPOLIA_RPC_URL=https://eth-sepolia.g.alchemy.com/v2/your_url
ETHERSCAN_API_KEY=YOUR_KEY
```
Don't forget to `source .env`, and then `make deploy`.
